Add Project in IntelliJ Idea by opening the directory AngryBirds in Intellij Idea as Open Existing Project
To Execute the Project, Just Go to the directory in the folder
lwjgl3 -- > src -- > main -- > java -- > com.unreal.angrybirds.lwjgl3 -- > Lwjgl3Launcher (Open)

In Lwjgl3Launcher.java file Execute the File in Intellij By using the Run Icon.
